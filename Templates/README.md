# Templates

Reusable building blocks. Claude should suggest pulling from here before creating anything from scratch.

## Structure

```
Templates/
├── prompts/    # Reusable Claude prompt patterns
├── docs/       # PRDs, briefs, reports, meeting notes
├── code/       # Scaffolds, boilerplate, file structures
└── comms/      # Emails, proposals, social posts, messages
```

## How to Use

1. Browse the relevant subfolder for an existing template.
2. Copy and fill in the `[PLACEHOLDER]` fields.
3. If you create something reusable from scratch, save it back here.

## How to Add a Template

Name files descriptively: `[type]-[use-case].md`
Examples: `prompt-code-review.md`, `doc-prd.md`, `comms-cold-email.md`
