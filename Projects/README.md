# Projects — Master System

## How to Launch Any New Project (30 seconds)

1. Create folder: `Projects/[software|business|personal]/[project-name]/`
2. Copy `brief-template.md` into it and fill the 6 fields
3. Tell Claude: "New project: [name] — [one sentence goal]"

## Project Brief Template

Copy into every new project as `brief.md`:

```
**Project Name:** 
**Type:** software / business / personal
**One-Sentence Goal:** 
**Success Looks Like (measurable):** 
**Hard Constraints / Must-Nots:** 
**Current State (what already exists):** 
**First Action I Want:** 
**Priority:** High / Medium / Low
**Deadline (if any):**
```

## Structure

```
Projects/
├── software/    # Apps, tools, scripts, codebases
├── business/    # Strategy, plans, SOPs, pitches
└── personal/    # Learning goals, side projects, personal productivity
```
